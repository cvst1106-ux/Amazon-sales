# Amazon-sales
## Análisis estadístico del Catálogo y Estrategia de Precios de Amazon
### Planteamiento del problema 
La compañía “Amazon” fue fundada en 1994 por Jeff Bezos, con el tiempo ha evolucionado y se ha convertido en líder del comercio electrónico a nivel global, ofreciendo una amplia gama de productos y servicios, estos van desde ropa, libros, productos para hogar, productos electrónicos de todo tipo, entre otras categorías. 
El catálogo de Amazon presenta una gran heterogeneidad (por su variedad de productos), lo que significa que no existe un patrón definido de cómo se definen los precios, ni de cómo se estructuran internamente las tasas de descuentos; no obstante, no es evidente si estos responden a factores aleatorios, a una relación sistemática entre el precio y el descuento, o si responden a una estrategia comercial definida. Detrás de cada descuento, pueden haber diferentes intenciones, como la de atraer clientes, posicionar categorías, o vaciar inventario.
En relación con lo expuesto anteriormente, la ausencia de un análisis que relacione estas variables genera incertidumbre en vendedores y consumidores. En este sentido, se plantean las siguientes interrogantes:
* ¿Existen categorías donde los descuentos son más altos que el resto del catálogo y, si éstas existen, hay alguna relación entre el precio original y su porcentaje de descuento? 
* ¿La distribución de los productos está equilibrada entre categorías o, el catálogo depende de pocas categorías dominantes? 
* ¿Dentro de las categorías, conviven productos de distintos rangos de precios o existe una homogeneidad entre ellos? 
#### Objetivos generales 
Describir la estructura del inventario de Amazon y analizar su estrategia de precios según las diferentes categorias de productos, identificando las brechas de precios entre productos similares.
#### Objetivos específicos
* 1. Analizar las diferencias entre productos similares, y la relación de precios entre el precio original y su porcentaje descuento en categorías con precios más altos.
* 2. Identificar el número de productos existentes en cada categoría para determinar si el catálogo está distribuido equilibradamente o está concentrado en pocas categorías dominantes.
* 3. Evaluar  la variabilidad de precios dentro de cada categoría para determinar si existen diferencias significativas entre productos de distintos rangos o si hay homogeneidad.
